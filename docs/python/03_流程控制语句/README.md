## 流程控制语句
 
### if 
```
if 条件表达式 : 
          代码块
```
### if-else
```
   if 条件表达式 :
       代码块
   else :
       代码块
```
### if-elif-else
```
   if 条件表达式 :
       代码块
   elif :
       代码块
   else : 
       代码块    
```
> break可以用来立即退出循环语句（包括else）
> continue可以用来跳过当次循环

### while
```
    while True :
         print(True)
     else :
        print(False)
```
